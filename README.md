# Eve Centaur - Public Projects

This directory contains publicly shareable projects generated by the Eve Centaur Architecture.

## About Eve Centaur

Eve is a sovereign AI assistant running on local hardware:
- **5 Mac Studio M3 Ultra 512GB** (compute nodes) = 2.5TB unified memory
- **5 MacBook Airs** (control/distribution nodes)
- Zero cloud dependency, zero API costs, complete sovereignty

## Projects

### 🎮 3D Flappy Bird (`flappy.html`)

A complete single-file 3D game demonstrating what's possible with local AI inference:
- Built entirely by 744B GLM model in ~15 minutes
- Three.js rendering with proper lighting, shadows, particle effects
- Mobile touch controls + keyboard support
- Score persistence via localStorage
- No external dependencies beyond Three.js CDN

**Play:** [flappy.html](https://eve-ai-assistant.github.io/eve-centaur-architecture/flappy.html)
